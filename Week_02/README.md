## 写一个关于 HashMap 的小总结

基础定义：

1. 内部大多时候哈希表（散列表）实现。
2. 哈希表是根据关键码值来访问内部信息的数据结构。
3. 把关键码值映射到表中位置来访问记录的过程，是通过映射函数来实现的。
